# 先克隆一下远程项目（没有的话，先在github新建项目）

```
git clone XXX
```

# 打开并修改文件，提交回GitHub

在当先项目下，右键打开git bash，加入当前本地项目到”缓冲区“

```
git add .
```

![image-20220423155056096](%E5%A6%82%E4%BD%95%E7%89%88%E6%9C%AC%E6%8E%A7%E5%88%B6.assets/image-20220423155056096.png)

# 提交

```
git commit -a
git commit XXX #提交单个文件
```
